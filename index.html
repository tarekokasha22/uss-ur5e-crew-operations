You are observing the GIU Robotics USS UR5e simulation: ROS2 Humble, MoveIt2, voice stack, and a crew that definitely has opinions.

        
Deck & rooms

        
Each cell is a compartment with a nameplate (icon + title + subtitle). Crew start in their home rooms; the robot patrols on a fixed route through engineering, lab, voice booth, mess, and cargo.

        
Crew

        
Hover a character for a live line from their file. Click to open the classified dossier. Decor in each room has tooltips — explore the set dressing.

Panels

        
Mission log — System and ROS2 style events. Crew comms — Radio chatter, scripted scenes by ship hour, room banter, and robot lines. Ship status ticker — Rotating mission + flavor headlines.

        
Time & cycle

        
Ship time advances slowly in-universe (not wall clock). Day / night follows local hour for the viewport tint (deep night 02:00–05:00).
